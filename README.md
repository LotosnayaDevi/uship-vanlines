# Uship Van Lines Inc - Moving Company Website

A modern, JeskoJets-inspired landing page for a long-distance moving company.

## Features

- **Modern Design**: Inspired by the award-winning JeskoJets website
- **Responsive**: Mobile-first design that works on all devices
- **Animated**: Smooth scroll animations and interactive elements
- **Fast**: Static HTML with Tailwind CSS CDN for quick loading
- **SEO Ready**: Semantic HTML structure

## Sections

1. **Header** - Sticky navigation with trust elements (DOT/MC numbers, ratings)
2. **Hero** - Immersive full-screen section with animated gradient background
3. **Quote Form** - Interactive form for instant quotes
4. **How It Works** - 4-step process visualization
5. **CTA Section** - Call-to-action with contact options
6. **Footer** - Company info, contact details, and disclaimer

## Technologies

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Outfit)

## Deployment

This site is deployed on Vercel. Simply push to the main branch for automatic deployments.
